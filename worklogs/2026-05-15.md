**Focus:** Full Wazuh stack deployment on Apple Silicon

## What I Did
- Created Ubuntu target VM (Ubuntu 24.04 ARM64) in UTM

## Alerts / Events Observed
- First alert fired: rule 5710 — SSH attempt with non-existent user
- Agent showing Active within 60 seconds of enrollment

## Key Finding
The official Wazuh installer rejects ARM64 with an architecture error. All four components had to be installed manually via apt — undocumented in the official guide. Five ARM64-specific bugs found and fixed across installation.
