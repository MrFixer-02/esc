# SOC Lab Worklog — 2026-05-15

**Phase:** A — Foundation
**Focus:** Full Wazuh stack deployment on Apple Silicon

## What I Worked On
- Created Ubuntu target VM (Ubuntu 24.04 ARM64) in UTM
- Created Wazuh server VM (Ubuntu 22.04 ARM64) in UTM
- Installed Wazuh Indexer, Manager, Filebeat, and Dashboard manually via apt
- Verified dashboard accessible at https://192.168.64.4
- Enrolled ubuntu-target as first Wazuh agent

## Alerts / Events Observed
- First alert: rule 5710 — SSH attempt with non-existent user
- Agent showing Active in dashboard within 60 seconds of enrollment

## Key Finding
The official Wazuh installer (`wazuh-install.sh`) rejects ARM64 with an architecture error. All four components had to be installed manually. This is undocumented in the official guide.
